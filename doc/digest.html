Class:jCastle.digest#

해쉬 함수는 임의의 길이를 갖는 메시지를 입력 받아 고정된 길이의 해쉬값을 출력하는 함수입니다. 암호 알고리즘에는 키가 사용되지만, 해쉬 함수는 키를 사용하지 않으므로 같은 입력에 대해서는 항상 같은 출력이 나오게 됩니다. 이러한 함수를 사용하는 목적은 입력 메시지에 대한 변경할 수 없는 증거값을 뽑아냄으로서 메시지의 오류나 변조를 탐지할 수 있는 무결성을 제공하는 목적으로 주로 사용됩니다.

new jCastle.digest([hash_name])#

jCastle.digest 오브젝트를 생성합니다.

  • hash_name <string>생성된 객체에 사용할 해쉬 알고리듬 이름.
    이 변수에 null값이 올 수 있습니다. 만일 null값이 주어지면 jCastle.digest::start 혹은 jCastle.digest::encrypt에 주어지는 변수 값에 algoName이라는 프로퍼티와 그 값이 주어져야 합니다.

Static method: jCastle.digest.create([hash_name])#

jCastle.digest 오브젝트를 생성합니다. new jCastle.digest()와 동일합니다.

  • hash_name <string>생성된 객체에 사용할 해쉬 알고리듬 이름.
  • Return Value: jCastle.digest 오브젝트를 생성하여 리턴합니다.
var hash_name = "SHA-512";
var md = jCastle.digest.create(hash_name);
var hash = md.digest("The quick fox jumps over the lazy dog!");
console.log(hash.toString("hex"));

Static method: jCastle.digest.start(options)#

jCastle.digest 오브젝트를 생성한 후 digest.start() 메소드를 실행합니다.

  • options <Object> 해쉬를 생성하기 위한 변수들을 담은 오브젝트. digest.start()를 참조하세요.
  • Return Value: jCastle.digest 오브젝트를 생성하여 digest.start()를 수행한 후 동일 오브젝트를 리턴합니다.

Static method: .jCastle.digest.digest(message, options) #

주어진 데이터를 가지고 해쉬를 생성하고 그 결과 값을 리턴합니다. options 오브젝트에는 반드시 algoName이 주어져야 합니다.

  • message <Buffer> 해쉬화할 입력 데이터. 만일 <string>으로 주어지면 자동으로 <Buffer>로 변환됩니다.
  • options <Object> 해쉬 생성에 필요한 변수들을 담은 오브젝트로 digest.start의 options 변수를 참조하세요.
  • Return Value: 해쉬를 <Buffer>로 리턴합니다.

digest.start(options)#

해쉬를 생성하기 위한 파라미터를 받아 프로세스를 준비합니다.

  • options <Object> 해쉬 생성을 위한 변수 값들을 담은 오브젝트
    • algoName <string> 해쉬 알고리듬 이름.
      객체 생성시 해쉬 알고리듬을 지정하지 않았을 경우 필요합니다.
    • rounds <Integer> 해쉬 생성 프로세스 라운드 값을 지정합니다.
      haval(3|4|5)과 tiger(3|4)는 라운드 값을 지정하도록 되어 있습니다.
    • outputBits <Integer> 해쉬 결과 값 리턴시 스트링 길이를 지정합니다.
      sha3 패밀리와 shake 및 skein은 스트링 길이 값을 지정할 수 있습니다. outputBits의 경우 비트 길이를 의미합니다.
    • outputSize <Integer> 해쉬 결과 값 리턴시 스트링 길이를 지정합니다.
      sha3 패밀리와 shake 및 skein은 스트링 길이 값을 지정할 수 있습니다. outputSize의 경우 바이트 길이를 의미합니다.
  • Return Value: 오브젝트 자신을 리턴합니다. 
var hash_name = "SHA-512";
var md = new jCastle.digest(hash_name);
md.start();
md.update("The quick fox jumps");
md.update(" over the lazy dog!");
var hash = md.finalize().toString('hex');
console.log(hash);

digest.update(message)#

메시지 해쉬 프로세스를 실행합니다.

  • message <Buffer> 해쉬를 생성할 입력 데이터. 만일 <string>으로 주어지면 자동으로 <Buffer>로 변환됩니다.
  • Return Value: 오브젝트 자신을 리턴합니다.

digest.finalize([message])#

프로세스를 종료하고 결과를 리턴합니다.

  • message <Buffer> 해쉬를 생성할 입력 데이터. 만일 <string>으로 주어지면 자동으로 <Buffer>로 변환됩니다.
  • Return Value: 생성된 해쉬를 <Buffer> 데이터를 리턴합니다.

digest.digest(message [, options])#

메시지 해쉬를 생성합니다.

  • message <Buffer> 해쉬를 생성할 입력 데이터. 만일 <string>으로 주어지면 자동으로 <Buffer>로 변환됩니다.
  • options <Object> 해쉬 생성에 필요한 변수들을 담은 오브젝트로 digest.start의 options 변수를 참조하세요.
  • Return Value: 생성된 해쉬를 <Buffer> 데이터를 리턴합니다.
