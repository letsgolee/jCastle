Class:jCastle.ecdh#

ECDH는 타원곡선 알고리듬(Elliptic Curve Cryptographic System)을 이용하여 Diffie-Hellman 키 교환 방식을 구현한 알고리듬입니다. 일반적으로 ECDSA는 RSA보다 키 길이가 작으면서도 암호화 성능이 뛰어나기 때문에 DH보다는ECDH를 키 교환 알고리듬으로 더 많이 사용하는 추세입니다.

new jCastle.ecdh(ecdsa)#

jCastle.ecdh 오브젝트를 생성합니다.

  • ecdsa <Object> jCastle.pki.ecdsa 객체 혹은 jCastle.pki 객체(이 경우 ECDSA로 초기화되어 있어야 합니다.). 이 값이 주어지면 ecdh.init(ecdsa)를 실행합니다.

Static method: jCastle.ecdh.create(ecdsa)#

jCastle.ecdh 오브젝트를 생성합니다. new jCastle.ecdh()와 동일합니다.

  • ecdsa <Object> jCastle.pki.ecdsa 객체 혹은 jCastle.pki 객체(이 경우 ECDSA로 초기화되어 있어야 합니다.). 이 값이 주어지면 ecdh.init(ecdsa)를 실행합니다.
  • Return Value: jCastle.ecdh 오브젝트를 생성하여 리턴합니다.

Static method: jCastle.ecdh.init(ecdsa)#

jCastle.ecdh 오브젝트를 생성한 후 ecdh.init()를 실행합니다. 자세한 내용은 ecdh.init()을 참조해 주세요.

  • ecdsa <Object> jCastle.pki.ecdsa 객체 혹은 jCastle.pki 객체(이 경우 ECDSA로 초기화되어 있어야 합니다.). 이 값이 주어지면 ecdh.init(ecdsa)를 실행합니다.
  • Return Value: jCastle.ecdh 오브젝트를 생성하여 ecdh.init()를 실행 후 리턴합니다.

ecdh.reset() #

입력되었던 모든 데이터들을 초기화 합니다.

  • Return Value: 오브젝트 자신을 리턴합니다.

ecdh.init(ecdsa)#

jCastle.ecdh 객체를 초기화하고 주어진 알고리듬을 등록합니다.

  • ecdsa <Object> jCastle.pki.ecdsa 객체 혹은 jCastle.pki 객체(이 경우 ECDSA로 초기화되어 있어야 합니다.). 이 값이 주어지면 ecdh.init(ecdsa)를 실행합니다.
  • Return Value: 객체 자신을 리턴합니다.

ecdh.getPublicKey(format) #

등록된 jCastle.pki.ecdsa의 공개키 y를 내보냅니다.

  • format <string> 공개키 포맷 형식. 기본 값은 object입니다. 올 수 있는 값은 hex, jwt, object, buffer, base64 등이 있습니다.
  • Return Value: 공개키를 format 값에 따라 내보냅니다.

ecdh.computeSecret(party_pubkey, additional) #

상대방의 공개키를 받아 비밀값을 계산합니다.

  • party_pubkey <Object> | <Buffer> | <string> 공개키 y. 만일 객체로 오는 경우 JWK 형태로 주어져야 합니다. 스트링은 헥스 스트링만 허용합니다.
  • additional <BigInteger> | <Buffer> 추가 계산용 값. 상대방과 동일하게 가지고 있는 값으로 추적을 더 어렵게 만들 수 있습니다.
  • Return Value: 비밀값을 <Buffer> 형태로 내보냅니다.

ecdh.calculateAgreement(party_pubkey, additional) #

상대방의 공개키를 받아 합의된 비밀값을 계산합니다. SEC 1: Elliptic Curve Cryptography를 참조하세요.

  • party_pubkey <Object> | <Buffer> | <string> 공개키 y. 만일 객체로 오는 경우 JWK 형태로 주어져야 합니다. 스트링은 헥스 스트링만 허용합니다.
  • additional <BigInteger> | <Buffer> 추가 계산용 값. 상대방과 동일하게 가지고 있는 값으로 추적을 더 어렵게 만들 수 있습니다.
  • Return Value: 비밀값을 <Buffer> 형태로 내보냅니다.

ecdh.calculateMQVAgreement(ephemeral_prvkey, party_pubkey, ephemeral_party_pubkey) #

상대방의 공개키 및 Short-time 공개키를 받아 Short-time용 개인키를 가지고 합의된 MQV 비밀값을 계산합니다. SEC 1: Elliptic Curve Cryptography의 6.2를 참조하세요.

  • ephemeral_prvkey <BigInteger> | <Buffer> Short-time용 개인키. Node.js에서 기본 지원하는 <BigInt>는 아직 지원하지 않습니다. <BigInteger>는 jCastle.math.bigInteger로 등록되어 있습니다.
  • party_pubkey <Buffer> | <string> 공개키 y. 스트링은 헥스 스트링만 허용합니다.
  • ephemeral_party_pubkey <Buffer> | <string> Short-time용 공개키. 스트링은 헥스 스트링만 허용합니다.
  • Return Value: 비밀값을 <Buffer> 형태로 내보냅니다.

    var alice = new jCastle.pki('ECDSA');
    var bob = new jCastle.pki('ECDSA');
    
    alice.parseParameters(brainpoolP512t1_pem);
    bob.parseParameters(brainpoolP512t1_pem);
    
    // Now allice and Bob both has the same parameters. now generate public & private key...

    alice.generateKeypair();
    bob.generateKeypair();
    
    // get public keys...
    
    var alice_pubkey = alice.getPublicKey().encodePoint();
    var bob_pubkey = bob.getPublicKey().encodePoint();

    //----------------------------------------------------------------------------------------------------
    
    
    // Compute secret...
    
    var alice_ecdh = new jCastle.ecdh().init(alice);
    var bob_ecdh = new jCastle.ecdh().init(bob);
    
    var alice_secret = alice_ecdh.computeSecret(bob_pubkey);
    var bob_secret = bob_ecdh.computeSecret(alice_pubkey);

    // Both secrets should be equal
    
    assert.ok(alice_secret.equals(bob_secret), "DiffieHellman Key Agreement with ECDSA Parameters(ECDH) Test");
   

    //----------------------------------------------------------------------------------------------------
    
    // creating ephemeral private key
    var prng = jCastle.prng.create();
    var alice_ephemeral_privkey = BigInteger.random(alice.getBitLength(), prng);
    var bob_ephemeral_privkey = BigInteger.random(bob.getBitLength(), prng);

    // creating ephemeral public key
    // var alice_ephemeral_pubkey = alice.pkiObject.ecInfo.G.multiply(alice_ephemeral_privkey);
    // var bob_ephemeral_pubkey = bob.pkiObject.ecInfo.G.multiply(bob_ephemeral_privkey);
    var alice_ephemeral_pubkey = alice.getEphemeralPublicKey(alice_ephemeral_privkey);
    var bob_ephemeral_pubkey = bob.getEphemeralPublicKey(bob_ephemeral_privkey);

    var alice_ecdh = jCastle.ecdh.create().init(alice);
    var bob_ecdh = jCastle.ecdh.create().init(bob);
    
    var zz = alice_ecdh.calculateMQVAgreement(alice_ephemeral_privkey, bob_pubkey, bob_ephemeral_pubkey);
    var zz1 = bob_ecdh.calculateMQVAgreement(bob_ephemeral_privkey, alice_pubkey, alice_ephemeral_pubkey);
    
    assert.ok(zz1.equals(zz), 'MQV Key Agreement with ECDSA Test');
   
});
