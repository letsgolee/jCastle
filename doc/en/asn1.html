Class:jCastle.asn1#

ASN.1은 Abstract Syntax Notation #1을 말하며, 데이터 구조를 기술하는데 사용되는 언어를 말합니다. 개인키 보관 및 인증서 보관은 PEM 형식의 스트링을 사용하는데 PEM은 근본적으로 BER 및 DER 형식의 스트링입니다. jCastle.asn1은 이 스트링을 사용하기 위해 JSON 형식으로 전환하는 객체입니다.

new jCastle.asn1()#

jCastle.asn1 오브젝트를 생성합니다.

Static method: jCastle.asn1.create()#

jCastle.asn1 오브젝트를 생성합니다. new jCastle.asn1()와 동일합니다.

  • Return Value: jCastle.asn1 오브젝트를 생성하여 리턴합니다.

Static method: jCastle.asn1.parse(data)#

jCastle.asn1 오브젝트를 생성한 후 asn1.parse()를 실행합니다.

  • data <string> | <Buffer> 파싱할 ASN#1 형식의 스트링 혹은 Buffer 데이터.
  • Return Value: 데이터를 파싱하여 그 결과를 <Object> 형식으로 리턴합니다.

Static method: jCastle.asn1.getDER(schema)#

jCastle.asn1 오브젝트를 생성한 후 asn1.getDER(schema)를 실행합니다. 자세한 내용은 asn1.getDER()를 참조하세요.

  • schema <Object> ASN#1 형식으로 변환할 객체.
  • Return Value: 객체를 변환하여 그 결과를 <string> 형식으로 리턴합니다.

Static method: jCastle.asn1.isSequence(obj)#

해당 객체가 ASN#1 Sequence에 해당하는지 체크합니다.

  • obj <Object> ASN#1 형식의 Schema 객체.
  • Return Value: 객체가 Sequence에 해당하면 true, 그렇지 않으면 false를 <Boolean> 형식으로 리턴합니다.

Static method: jCastle.asn1.isAsn1Format(data)#

해당 객체가 ASN#1 문법에 해당하는 객체인지 체크합니다. 이 함수는 Recursive로 작동합니다.

  • data <Object> ASN#1 형식의 Schema 객체.
  • Return Value: 객체가 ASN#1 문법에 해당하면 true, 그렇지 않으면 false를 <Boolean> 형식으로 리턴합니다.

asn1.parse(data)#

주어진 데이터를 파싱하여 ASN#1 문법의 특성을 갖는 객체를 생성합니다. 이 객체를 schema 객체로 부르겠습니다.

  • data <string> | <Buffer> 파싱할 ASN#1 형식의 스트링 혹은 Buffer 데이터.
  • Return Value: 데이터를 파싱하여 그 결과를 <Object> 형식으로 리턴합니다.

asn1.getDER(schema, ber_encoding = false)#

schema 객체를 이용하여 ASN#1 문법의 DER 스트링을 생성합니다.

  • schema <Object> ASN#1 형식으로 변환할 객체.
  • ber_encoding <Boolean> BER 문법으로 생성하고자 할 때
  • Return Value: 객체를 변환하여 그 결과를 <string> 형식으로 리턴합니다.

asn1.getBER(schema)#

schema 객체를 이용하여 ASN#1 문법의 BER 스트링을 생성합니다.

  • schema <Object> ASN#1 형식으로 변환할 객체.
  • Return Value: 객체를 변환하여 그 결과를 <string> 형식으로 리턴합니다.

asn1.getBuffer(schema, ber_encoding = false)#

schema 객체를 이용하여 ASN#1 문법의 DER을 생성 후 버퍼로 리턴합니다.

  • schema <Object> ASN#1 형식으로 변환할 객체.
  • ber_encoding <Boolean> BER 문법으로 생성하고자 할 때
  • Return Value: 객체를 변환하여 그 결과를 <Buffer> 형식으로 리턴합니다.
