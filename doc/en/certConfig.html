Class:jCastle.certConfig#

OpenSSL의 경우 인증서 발급 및 관리를 위해 openssl.cnf라는 환경설정 파일을 사용합니다. jCastle.CertConfig은 환경설정 스트링을 분석하는 툴입니다. 이 툴을 이용하여 인증성의 생성시 사용합니다. 자바스크립트의 특성 및 브라우져에서 동작하는 것을 원칙으로 jCastle이 개발되었기 때문에 CA_default에 등록된 파일을 읽어오는 부분은 구현되어 있지 않습니다. 하지만 인증서의 Extensions 설정에 대한 부분은 대부분 구현되어 있습니다.

new jCastle.certConfig()#

jCastle.certConfig 오브젝트를 생성합니다.

Static method: jCastle.certConfig.create()#

jCastle.certConfig 오브젝트를 생성합니다. new jCastle.certConfig()와 동일합니다.

  • Return Value: jCastle.certConfig 오브젝트를 생성하여 리턴합니다.

Static method: jCastle.certConfig.parse(cnf_content, env_variables)#

jCastle.certConfig 오브젝트를 생성한 후 certConfig.parse(data)를 실행합니다. 자세한 내용은 certConfig.parse()를 참조하세요.

  • cnf_content <string> | <Buffer> 파싱할 OpenSSL의 환경파일 내용 스트링 혹은 Buffer 데이터.
  • env_variables <Object> 환경 변수들을 담은 객체
  • Return Value: 데이터를 파싱하여 그 결과를 <Object> 형식으로 리턴합니다.

  • cnf_content <string> | <Buffer> 파싱할 OpenSSL의 환경파일 내용 스트링 혹은 Buffer 데이터.
  • env_variables <Object> 환경 변수들을 담은 객체
  • Return Value: 데이터를 파싱하여 그 결과를 <Object> 형식으로 리턴합니다.
