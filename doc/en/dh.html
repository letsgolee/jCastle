Class:jCastle.dh#

디피-헬만 키 교환(Diffie–Hellman key exchange)은 암호 키를 교환하는 하나의 방법으로, 상대방과 암호화되지 않은 통신망을 통해 공통의 비밀 키를 공유할 수 있도록 합니다.

디피-헬만 키 교환은 소수 p와 g를 적절하게 고르면 도청에 대해 안전한 것으로 알려져 있습니다. 도청자 이브가 비밀키를 얻어내기 위해서는 앨리스와 밥 사이의 통신에서 도청할 수 있는 정보인 g^a와 g^b로부터 g^{ab}를 구해야 합니다.

이 문제를 디피-헬만 문제(Diffie-Hellman problem)로 부르며, 이 문제를 푸는 효율적인 알고리즘은 아직까지 알려지지 않았습니다. 이산 로그 문제를 효율적으로 풀 수 있을 경우 디피-헬만 문제 또한 효율적으로 풀 수 있지만, 그 역이 참인지는 알려지지 않았습니다.

new jCastle.dh(algo)#

jCastle.dh 오브젝트를 생성합니다.

  • algo <Object> jCastle.pki.* 객체(dsa, kcdsa 또는 elGamal) 혹은 jCastle.pki 객체. 이 값이 주어지면 dh.init(algo)를 실행합니다.

Static method: jCastle.dh.create(algo)#

jCastle.dh 오브젝트를 생성합니다. new jCastle.dh()와 동일합니다.

  • algo <Object> jCastle.pki.* 객체(dsa, kcdsa 또는 elGamal) 혹은 jCastle.pki 객체. 이 값이 주어지면 dh.init(algo)를 실행합니다.
  • Return Value: jCastle.dh 오브젝트를 생성하여 리턴합니다.

Static method: jCastle.dh.init(algo)#

jCastle.dh 오브젝트를 생성한 후 dh.init()를 실행합니다. 자세한 내용은 dh.init()을 참조해 주세요.

  • algo <Object> jCastle.pki.* 객체(dsa, kcdsa 또는 elGamal) 혹은 jCastle.pki 객체. 이 값이 주어지면 dh.init(algo)를 실행합니다.
  • Return Value: jCastle.dh 오브젝트를 생성하여 dh.init()를 실행 후 리턴합니다.

dh.reset() #

입력되었던 모든 데이터들을 초기화 합니다.

  • Return Value: 오브젝트 자신을 리턴합니다.

dh.init(algo)#

jCastle.dh 객체를 초기화하고 주어진 알고리듬을 등록합니다.

  • algo <Object> jCastle.pki.* 객체(dsa, kcdsa 또는 elGamal) 혹은 jCastle.pki 객체.
  • Return Value: 객체 자신을 리턴합니다.

dh.getPublicKey(format) #

등록된 jCastle.pki의 공개키 y를 내보냅니다.

  • format <string> 공개키 포맷 형식. 기본 값은 object입니다. 올 수 있는 값은 hex, jwt, object, buffer, uint8array, base64 등이 있습니다.
  • Return Value: 공개키를 format 값에 따라 내보냅니다.

dh.computeSecret(other_pubkey, additional) #

상대방의 공개키를 받아 비밀값을 계산합니다.

  • other_pubkey <BigInteger> | <Buffer> 상대방의 공개키 y.
  • additional <BigInteger> | <Buffer> 추가 계산용 값. 상대방과 동일하게 가지고 있는 값으로 추적을 더 어렵게 만들 수 있습니다.
  • Return Value: 비밀값을 <Buffer> 형태로 내보냅니다.
QUnit.test("DH Test", function(assert) {
  //----------------------------------------------------------------------------------------------------
    
    
  var params = jCastle.pki.dsa.getPredefinedParameters(4096);
    
  var alice = new jCastle.pki('DSA');
  var bob = new jCastle.pki('DSA');

  alice.setParameters(params);
  bob.setParameters(params);

  // Now allice and Bob both has the same parameters. now generate public & private key...

  alice.generateKeypair();
  bob.generateKeypair();

  // Exchange public keys...

  var alice_pubkey = alice.getPublicKey();
  var bob_pubkey = bob.getPublicKey();

  // Compute secret...

  var alice_dh = new jCastle.dh().init(alice);
  var bob_dh = new jCastle.dh().init(bob);

  var alice_secret = alice_dh.computeSecret(bob_pubkey).toString('hex');
  var bob_secret = bob_dh.computeSecret(alice_pubkey).toString('hex');

  // Both secrets should be equal

  assert.equal(alice_secret , bob_secret, "DiffieHellman Key Agreement with DSA Parameters Test");
});
