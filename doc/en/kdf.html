Object:jCastle.kdf#

Static method: jCastle.kdf.pkcs5DeriveKey(password, salt, len, hash_algo = 'md5')#

PKCS#5 키 생성 함수

  • password <Buffer> 비밀번호
  • salt <Buffer> 랜덤 키 생성을 위한 솔트 값
  • len <Integer> 키 길이
  • hash_algo <string> 키 생성시 사용할 해쉬 명. 기본 값은 md5입니다.
  • Return Value: 생성된 키를 <Buffer>로 리턴합니다.

Static method: jCastle.kdf.ansX963DeriveKey(hash_algo, dklen, z, sharedInfo)#

ANS X9.63 키 생성 함수. SEC 1: Elliptic Curve Cryptography를 참조하세요.

  • hash_algo <string> 키 생성시 사용할 해쉬 명
  • dklen <Integer> 키 길이
  • z <Buffer> 키 생성을 위한 Z 값
  • sharedInfo <Buffer> 키 생성을 위한 sharedInfo 값
  • Return Value: 생성된 키를 <Buffer>로 리턴합니다.

Static method: jCastle.kdf.singlestepKDF(Z, keylen, other_info, hash_algo = 'sha-256', hash_type = 'hash', salt)#

NIST의 Single-step 키 생성 함수. Recommendation for Pair-Wise Key Establishment Schemes Using Discrete Logarithm Cryptography 를 참조하세요.

  • Z <Buffer> 키 생성을 위한 Z 값
  • keylen <Integer> 키 길이
  • other_info <Buffer> 키 생성을 위한 OhterInfo 값
  • hash_algo <string> 키 생성시 사용할 해쉬 명. 기본 값은 sha-256입니다.
  • hash_type 해쉬 타입. hash와 hmac이 있습니다. 기본 값은 hash입니다.
  • salt <Buffer> 랜덤 키 생성을 위한 솔트 값
  • Return Value: 생성된 키를 <Buffer>로 리턴합니다.

Static method: jCastle.kdf.pkcs12DeriveKey(password, salt, iterations, id, len, hash_algo = 'sha-1'[, bmpPassword])#

PKCS#12 키 생성 함수. PKCS #12: Personal Information Exchange Syntax v1.1를 참조하세요.

  • password <Buffer> 비밀번호
  • salt <Buffer> 랜덤 키 생성을 위한 솔트 값
  • iterations <Integer> 해쉬 생성 반복 값
  • id <Integer> 해쉬 생성 ID. KEY 생성시는 1, IV 생성시는 2, MAC 생성시는 3 값.
  • len <Integer> 키 길이
  • hash_algo <string> 키 생성시 사용할 해쉬 명. 기본 값은 sha-1입니다.
  • bmpPassword <Buffer> BMP 타입 비밀번호. 사용하려면 password값을 null로 입력하세요.
  • Return Value: 생성된 키를 <Buffer>로 리턴합니다.
