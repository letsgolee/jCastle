Class:jCastle.keywrap#

키 싸기(Key Wrap) 알고리듬이란 암호 키를 캡슐화 (암호화)하도록 고안된 대칭 키 암호 알고리듬의 하나로서, 키 포장 알고리듬은 신뢰하지 않는 장치에 있는 키를 보호하거나, 신뢰되지 않은 네트워크 통신 구간에서의 키 전송을 하는 응용 프로그램에서 사용됩니다.

키 싸기 알고리듬은 기본 단위로 AES 알고리듬을 이용하나 블럭 사이즈 16을 지원하는 알고리듬이면 사용 가능합니다.

이 알고리듬의 목적은 키 전송에 있으나 꼭 키가 아니어도 무방합니다.

new jCastle.keywrap([algo_name])#

jCastle.keywrap 오브젝트를 생성합니다.

  • algo_name <string>생성된 객체에 사용할 알고리듬 이름. jCastle.keywrap의 기본으로 설정된 값은 aes-128입니다.

Static method: jCastle.keywrap.create([algo_name])#

jCastle.keywrap 오브젝트를 생성합니다. new jCastle.keywrap()와 동일합니다.

  • algo_name <string>생성된 객체에 사용할 알고리듬 이름.
  • Return Value: jCastle.keywrap 오브젝트를 생성하여 리턴합니다.

Static method: jCastle.keywrap.start(options)#

jCastle.keywrap 오브젝트를 생성한 후 keywrap.start() 메소드를 실행합니다.

  • options <Object> 암호화 혹은 복호화 과정을 수행하기 위한 변수들을 담은 오브젝트. keywrap.start()를 참조하세요.
  • Return Value: keywrap.mcrypt 오브젝트를 생성하여 keywrap.start()를 수행한 후 동일 오브젝트를 리턴합니다.

Static method: jCastle.keywrap.wrap(keydata, options) #

주어진 데이터를 가지고 키 감싸기 암호화 과정을 진행하고 그 결과 값을 리턴합니다. options 오브젝트에는 algoName이 올 수 있습니다.

  • keydata <Buffer> 암호화할 입력 데이터. 만일 <string>으로 주어지면 자동으로 <Buffer>로 변환됩니다.
  • options <Object> 암호화에 필요한 변수들을 담은 오브젝트로 keywrap.start의 options 변수를 참조하세요.
  • Return Value: 암호화한 결과로 <Buffer> 데이터를 리턴합니다.

Static method: jCastle.keywrap.unwrap(keydata, options) #

주어진 데이터를 가지고 키 감싸기 복호화 과정을 진행하고 그 결과 값을 리턴합니다. options 오브젝트에는 algoName이 올 수 있습니다.

  • keydata <Buffer> 복호화할 입력 데이터. 만일 <string>으로 주어지면 자동으로 <Buffer>로 변환됩니다.
  • options <Object> 복호화에 필요한 변수들을 담은 오브젝트로 keywrap.start의 options 변수를 참조하세요.
  • Return Value: 복호화한 결과로 <Buffer> 데이터를 리턴합니다.

Static method: jCastle.keywrap.getOID(algo)#

키 감싸기 알고리듬의 OID(Object ID)가 존재할 경우 해당 OID값을 리턴합니다.

  • algo <string> 베이스 알고리듬 이름.
  • Return Value: 키 감싸기 알고리듬의 OID를 <string>로 리턴합니다.

Static method: jCastle.keywrap.getWrapName(algo)#

키 감싸기 알고리듬의 등록된 명칭이 존재할 경우 해당 명칭을 리턴합니다.

  • algo <string> 베이스 알고리듬 이름.
  • Return Value: 키 감싸기 알고리듬의 명칭을 <string>로 리턴합니다.

Static method: jCastle.keywrap.getAlgoName(algo)#

키 감싸기 알고리듬의 베이스 알고리듬 명칭을 리턴합니다.

  • wrap_algo <string> 키 감싸기 알고리듬 이름.
  • Return Value: 베이스 알고리듬의 명칭을 <string>로 리턴합니다.

keywrap.start(options)#

키 감싸기 암호화 또는 복호화 과정을 시작합니다.

  • options <Object> 옵션 객체
    • algoName <string> 키 감싸기 암호화/복호화에 사용할 알고리즘 명칭
    • wrappingKey <Buffer> 암호화/복호화하기 위해 필요한 키로 만일 <string>으로 주어지면 자동으로 <Buffer>로 변환됩니다. 입력 스트링은 UTF-8 인코딩으로 간주되기 보다 binary 스트링으로 간주된다는 점을 유의하세요.
    • isEncryption <Boolean> 암호화 혹은 복호화인지에 대한 Boolean 변수.
  • Return Value: 오브젝트 자신을 리턴합니다.

암호화 혹은 복호화 할 데이터를 입력 데이터로 받습니다.

  • keydata <Buffer> 암호화 또는 복호화 할 입력 데이터. 만일 <string>으로 주어지면 자동으로 <Buffer>로 변환됩니다.
  • Return Value: 오브젝트 자신을 리턴합니다.

암호화 또는 복호화 과정을 멈추고 그 결과 값을 리턴합니다.

  • keydata <Buffer> 암호화 또는 복호화 할 입력 데이터. 만일 <string>으로 주어지면 자동으로 <Buffer>로 변환됩니다.
  • Return Value: 암호화 또는 복호화 한 결과로 <Buffer> 데이터를 리턴합니다.

keywrap.unwrap(keydata, options) #

주어진 데이터를 가지고 키 감싸기 복호화 과정을 진행하고 그 결과 값을 리턴합니다.

  • keydata <Buffer> 복호화할 입력 데이터. 만일 <string>으로 주어지면 자동으로 <Buffer>로 변환됩니다.
  • options <Object> 복호화에 필요한 변수들을 담은 오브젝트로 keywrap.start의 options 변수를 참조하세요.
  • Return Value: 복호화한 결과로 <Buffer> 데이터를 리턴합니다.
var kw = new jCastle.KeyWrap();
var keydata = Buffer.from(
  `00112233445566778899AABBCCDDEEFF000102030405060708090A0B0C0D0E0F`,
  "hex"
);
var wrapping_key = Buffer.from(
  `000102030405060708090A0B0C0D0E0F101112131415161718191A1B1C1D1E1F`,
  "hex"
);
var expected = Buffer.from(
  `28C9F404C4B810F4 CBCCB35CFB87F826 3F5786E2D80ED326 CBC7F0E71A99F43B FB988B9B7A02DD21`.replace(
    / /g,
    ""
  ),
  "hex"
);
var ct = kw.wrap({
  wrappingKey: wrapping_key,
  keyData: keydata,
});
console.log(ct.equals(expected));

var pt = kw.unwrap({
  wrappingKey: wrapping_key,
  keyData: ct,
});
console.log(pt.equals(keydata));
