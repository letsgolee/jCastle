MAC 모드 생성과정은 관련 지식을 필요로 하므로 jCastle.mac을 사용하는 것을 권해드립니다.

Static method: jCastle.mac.mode.create(mode)#

MAC 모드를 생성합니다.

  • mode <string> 모드 명칭
  • Return Value: 생성된 jCastle.mac.mode.* 모드 오브젝트를 리턴합니다.

Static method: jCastle.mac.mode.listModes()#

지원하는 MAC 모드 리스트를 리턴합니다.

  • Return Value: MAC 모드 리스트를 <Array>로 리턴합니다.

mode.init(algorithm, options)#

모드를 초기화합니다.

  • algorithm <Object> jCastle.mcrypt.algorithm.* 객체
  • options <Object> 모드 수행을 위한 변수들을 담은 오브젝트.

mode.process(input)#

블럭 단위로 모드를 진행합니다.

  • input <Buffer> 블럭 데이터

mode.finish([input])#

블럭 단위로 MAC코드 생성을 위한 모드를 진행하고 그 결과를 리턴합니다.

  • input <Buffer> 블럭 데이터
  • Return Value: 결과를 <Buffer>로 리턴합니다.
var text = "The quick fox jumps over the lazy dog!";
//
// jCastle.mac을 이용하여 MAC 코드 생성
//
var mac_name = "CMAC";
var mac = jCastle.mac.create(mac_name);
mac.start({
  key: key,
  algoName: 'aes-128'
});
mac.update(text);
var code = mac.finalize().toString('hex');
console.log(code);

//
// jCastle.mac.mode를 이용하여 MAC 코드 생성
// 
var block_size = jCastle.mcrypt.getAlgorithmInfo("aes-128").blockSize;
var buf = Buffer.from(text);
var pos = 0;

var algorithm = jCastle.mcrypt.getAlgorithm("aes-128");
var cmacMode = jCastle.mac.mode.create("cmac");
cmacMode.init(algorithm, {
  key: key
});

while (p < buf.length) {
  // block 단위로 데이터 처리
  cmacMode.process(buf.slice(p, block_size));
  p += block_size;
}

var code = cmacMode.finish();
console.log(code.toString('hex'));
