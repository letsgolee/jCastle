Static method: jCastle.mcrypt.mode.create(mode)#

Creates cipher mode.

  • mode <string> Mode name
  • Return Value: Returns the instance of jCastle.mcrypt.mode.*.

Static method: jCastle.mcrypt.mode.listModes()#

Gets the list of cipher mode.

  • Return Value: Returns the list as <Array>.

mode.init(algorithm, options)#

Initializes the mode process.

  • algorithm <Object> an instance of jCastle.mcrypt.algorithm.*.
  • options <Object> Options object. Refer to the above table for needed properties.
    • isEncryption <Boolean> Flag for direction of process. If encryption then true, otherwise false
    • blockSize <Integer> block size for the process
    • iv <Buffer> Initial Vector.
    • nonce <Buffer> Nonce value for AEAD(ccm, eax, gcm, cwc).
    • additionalData An additional data for <Buffer> AEAD(nonce를 지원하는 모드 및 poly1305).
    • ctsType <string> cts type
    • tweakKey <Buffer> xts The tweak key value
    • tweak <Buffer> Only used in xts mode.
    • dataUnitSerial <Buffer> | <Integer> | <string> Only used for xts mode
    • dataUnitLength <Integer> Only used for xts mode
    • tagSize <Integer> Tag size for AEAD.

mode.process(input)#

Processes with the block size.

  • input <Buffer> block data

mode.finish([input])#

Finishes the process and returns the result.

  • input <Buffer> block data
  • Return Value: Returns the result as <Buffer>.
var ciphertext = new jCastle.mcrypt().encrypt("This is the message", {
  algoName: "AES-256",
  key: key,
  nonce: nonce,
  mode: "GCM",
  isEncryption: true,
  padding: "pkcs7",
  additionalData: adata
});

jCastle.mcrypt.mode object can be used with each algorithm object and an encryption or decryption with block cipher mode can be processed.
However using directly with the cipher mode is not easy without the knowledage of each cipher modes.
jCastle.mcrypt.mode requires padding process if the mode needs padding of the string.
Without the knowledage of jCastle.mcrypt.mode, it is desirable to use new jCastle.mcrypt. 

var is_encryption = true; // encryption
var block_size = jCastle.mcrypt.getBlockSize("aes-256");

var algo = new jCastle.algorithm.aes("aes-256"); // key length is 32 bytes
algo.keySchedule(key, is_encryption);

var cbcMode = jCastle.mcrypt.mode.create("cbc");
cbcMode.init(algo, {
  iv: initial_vector,
  blockSize: block_size,
  isEncryption: is_encryption
});
var cipher_block = cbcMode.process(input_block);
cbcMode.finish();
