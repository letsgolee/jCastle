Class:jCastle.prng#

new jCastle.prng([seed, hashAlgo])#

jCastle.prng 오브젝트를 생성합니다.

  • seed <Buffer> | <string> Seed 값. 주어지지 않으면 랜덤 값을 사용합니다.
  • hashAlgo <string> 사용할 해쉬 알고리듬 명칭. 주어지지 않으면 내부적으로 arc4 엔진을 사용합니다.

new jCastle.secureRandom([seed, hashAlgo])#

jCastle.secureRandom 오브젝트를 생성합니다. jCastle.prng와 동일합니다. 차이가 있다면 hashAlgo값이 주어지지 않으면 sha-1이 기본이 됩니다.

  • seed <Buffer> | <string> Seed 값. 주어지지 않으면 랜덤 값을 사용합니다.
  • hashAlgo <string> 사용할 해쉬 알고리듬 명칭. 주어지지 않으면 sha-1 엔진을 사용합니다.

Static method: jCastle.prng.create([seed, hashAlgo]) | jCastle.secureRandom.create([seed, hashAlgo])#

jCastle.prng 오브젝트를 생성합니다. new jCastle.prng()와 동일합니다.

  • seed <Buffer> | <string> Seed 값. 주어지지 않으면 랜덤 값을 사용합니다.
  • hashAlgo <string> 사용할 해쉬 알고리듬 명칭. 주어지지 않으면 내부적으로 arc4 엔진을 사용합니다.
  • Return Value: jCastle.prng 오브젝트를 생성하여 리턴합니다.

jCastle.prng 오브젝트를 생성 후 prng.randomBytes()를 실행합니다.

  • size <Integer> 리턴할 바이트 갯수 크기
  • seed <Buffer> | <string> Seed 값. 주어지지 않으면 랜덤 값을 사용합니다.
  • hashAlgo <string> 사용할 해쉬 알고리듬 명칭. 주어지지 않으면 내부적으로 arc4 엔진을 사용합니다.
  • Return Value: 랜덤으로 생성한 값을 <Buffer>로 리턴합니다.

prng.init(seed, hashAlgo)#

객체를 주어진 값으로 초기화 합니다.

  • seed <Buffer> | <string> Seed 값. 주어지지 않으면 랜덤 값을 사용합니다.
  • hashAlgo <string> 사용할 해쉬 알고리듬 명칭. 주어지지 않으면 내부적으로 arc4 엔진을 사용합니다.
  • Return Value: 객체 자신을 리턴합니다.

prng.nextByte(nozero = false)#

랜덤으로 생성한 바이트 값을 리턴합니다.

  • nozero <Boolean> 0x00값을 리턴할지 여부를 정합니다. 기본 값은 false입니다.
  • Return Value: 랜덤으로 생성한 바이트 값을 <Integer>로 리턴합니다.

prng.nextBytes(size, nozero_first = false, nozero = false)#

랜덤으로 생성한 바이트 값을 리턴합니다.

  • size <Integer> 리턴할 바이트 갯수 크기
  • nozero_first <Boolean> 첫 값이 0x00을 수용할지 여부를 정합니다. 기본 값은 false입니다.
  • nozero <Boolean> 0x00을 수용할지 여부를 정합니다. 기본 값은 false입니다.
  • Return Value: 랜덤으로 생성한 바이트 배열을 <Buffer>로 리턴합니다.

prng.randomBytes(size, nozero_first = false, nozero = false)#

랜덤으로 생성한 바이트 값을 리턴합니다. prng.nextBytes()와 동일합니다.

QUnit.test("generating random numbers test", function(assert) {
    // default algorithm - arc4.
    // same seed should give the same result.
    
    var seed = "jCastle.net Jacob Lee 2022.04.14 pure javascript crypto library!";

    var prng1 = new jCastle.prng(seed);
    var prng2 = jCastle.prng.create(seed);
    var prng3 = jCastle.prng.create(seed+'!'); // this one should not give the same result.

    var a, b, c;

    for (var i = 0; i < 100; i++) {
        a = prng1.nextBytes(16);
        b = prng2.nextBytes(16);
        c = prng3.nextBytes(16);

        assert.ok(a.equals(b), 'prng with the same seed test');
        assert.ok(!a.equals(c), 'prng with different seed test');
    }

});

QUnit.test("generating test 2", function(assert) {
    // no seed given, then seed will be generated internally.
    // sha-1 algorithm is used.
    var prng = new jCastle.prng(null, 'sha-1');
    var t;

    for (var i = 0; i < 10000; i++) {
        t = prng.nextBytes(64, true, true); // first boolean - no zero first byte, second boolean - allows no zero at all.
    }
    for (var i = 0; i < t.length; i++) {
        assert.ok(t[i] != 0x00, "prng no zero test");
    }
});
