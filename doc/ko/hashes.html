new jCastle.algorithm.*(hash_name [, options]])#

jCastle.mcrypt.algorithm.* 해쉬 알고리즘 오브젝트를 생성합니다.

  • hash_name <string> 생성된 객체에 사용할 해쉬 알고리듬 이름.

algorithm.init([options]) #

프로세스를 초기화합니다.

  • options <Object> 프로세스 시작을 위한 변수들을 담은 오브젝트.
    • rounds <Integer> 타입으로 haval, tiger 해쉬 알고리듬에서 사용 가능
    • outputSize <Integer> 해쉬 결과 값리턴시 스트링 길이를 지정합니다.
      sha3 패밀리와 shake 및 skein은 스트링 길이 값을 지정할 수 있습니다. outputSize의 경우 바이트 길이를 의미합니다.
    • hmacKey <Buffer> 타입으로 skein 해쉬 알고리즘에서 사용

algorithm.process(input)#

블럭 단위로 프로세스를 진행합니다.

  • input <Buffer> 해쉬화할 블럭 데이터

algorithm.finish()#

해쉬화 결과를 리턴합니다.

  • Return Value: 해쉬화 결과를 <Buffer>로 리턴합니다.

algorithm.pad(input, pos)#

입력 데이터를 블럭 사이즈에 맞게 패딩처리를 합니다.

  • input <Buffer> 전체 입력 데이터
  • pos <Integer> 현재 데이터 처리 포지션
  • Return Value: 패딩처리된 전체 입력 데이터를 리턴합니다.

해쉬 알고리듬 객체를 직접 사용하여 해쉬를 생성할 수 있습니다. 그러나 직접 사용하는 것은 각 해쉬 알고리듬에 대한 지식이 없이는 사용하기 어렵습니다. 다음의 예는 직접 사용할 경우의 방식을 보여줍니다:

var algo = new jCastle.algorithm.sha256("sha-256");
var message = "The quick fox jumps over the lazy dog!";

// get block size
var block_size = jCastle.digest.getBlockSize("sha-256");

// string to buffer
var input = Buffer.from(message);

algo.init();

var pos = 0;

while (pos + block_size <= input.length) {
  algo.process(input.slice(pos, pos + block_size));
  pos += block_size;
}

// padding
algo.pad(input, pos);

while (pos < input.length) {
  algo.process(input.slice(pos, pos + block_size));
  pos += block_size;
}

var output = algo.finish();

// change to hex string
var hash = output.toString('hex');
console.log(hash);
